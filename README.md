# Cats-vs-Dogs-Image-Classification
A deep learning project that classifies images of cats and dogs using Convolutional Neural Networks with TensorFlow and Keras.

## Overview

This project is a **binary image classification system** that classifies images of **cats and dogs** using a **Convolutional Neural Network** built with **TensorFlow and Keras**.

The project demonstrates an end-to-end deep learning workflow, including data preprocessing, model training, and evaluation.

---

## Dataset

* **Dataset:** PetImages (Cats vs Dogs)
* **Classes:** Cat, Dog
* **Source:** https://www.kaggle.com/datasets/shaunthesheep/microsoft-catsvsdogs-dataset

The dataset contains some corrupted images, which were handled during preprocessing.

---

## Tech Stack

* Python
* TensorFlow / Keras
* CNN (Convolutional Neural Network)

---

## Model

* CNN built using Keras Sequential API
* Binary classification using **Sigmoid** activation
* Loss: **Binary Crossentropy**
* Optimizer: **Adam**
